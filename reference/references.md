# Insert references at this position

Insert references at this position

## Usage

``` r
references(appendix = FALSE, part = FALSE)
```

## Arguments

- appendix:

  Chapters after the references become the appendix.

- part:

  Include the appendix as a part (level 0) heading. Only relevant for
  pdf output.
