# Package index

## Output formats

- [`epub_book()`](https://inbo.github.io/INBOmd/reference/epub_book.md)
  : Style for e-book
- [`gitbook()`](https://inbo.github.io/INBOmd/reference/gitbook.md) :
  Output format for a web version of the report
- [`handouts()`](https://inbo.github.io/INBOmd/reference/handouts.md) :
  Use the handout version of slides with the INBO theme
- [`minutes()`](https://inbo.github.io/INBOmd/reference/minutes.md) :
  Create a minutes of a meeting with the INBO corporate identity
- [`mission()`](https://inbo.github.io/INBOmd/reference/mission.md) :
  Create a mission report with the INBO corporate identity
- [`pdf_report()`](https://inbo.github.io/INBOmd/reference/pdf_report.md)
  : Create a report with the Flemish corporate identity
- [`poster()`](https://inbo.github.io/INBOmd/reference/poster.md) :
  Create a poster with the INBO theme version 2015
- [`slides()`](https://inbo.github.io/INBOmd/reference/slides.md) : pdf
  slides with an INBO or Flanders theme

## RStudio Addins

- [`insert_block()`](https://inbo.github.io/INBOmd/reference/blocks.md)
  [`insert_exampleblock()`](https://inbo.github.io/INBOmd/reference/blocks.md)
  [`insert_alertblock()`](https://inbo.github.io/INBOmd/reference/blocks.md)
  :

  Insert a markdown block These functions will add template chunks into
  your `.Rmd` file. A markdown block will be display as a frame with a
  title in a different background. There are 3 types of blocks:
  `"block"`, `"exampleblock"` and `"alertblock"`.

## Utilities

- [`add_author()`](https://inbo.github.io/INBOmd/reference/add_author.md)
  : Add an author to the a report

- [`add_report_numbers()`](https://inbo.github.io/INBOmd/reference/add_report_numbers.md)
  : Add the report numbers to a bookdown report

- [`check_dependencies()`](https://inbo.github.io/INBOmd/reference/check_dependencies.md)
  : Check if the required dependencies of INBOmd are installed Missing
  dependencies will be installed automatically

- [`column_start()`](https://inbo.github.io/INBOmd/reference/column_start.md)
  : Calculate the starting position of an A0 column

- [`column_width()`](https://inbo.github.io/INBOmd/reference/column_width.md)
  : Calculate the width of an A0 column

- [`cover_info()`](https://inbo.github.io/INBOmd/reference/cover_info.md)
  : Create a cover.txt from the YAML header

- [`create_report()`](https://inbo.github.io/INBOmd/reference/create_report.md)
  : Create a template for an INBOmd bookdown report

- [`dyn_table()`](https://inbo.github.io/INBOmd/reference/dyn_table.md)
  : Use a different table style depending on the output format

- [`inbo_website()`](https://inbo.github.io/INBOmd/reference/inbo_website.md)
  : Render the report into a zip file for the INBO website

- [`prepare_cover()`](https://inbo.github.io/INBOmd/reference/prepare_cover.md)
  : Prepare a cover image from a PDF file

- [`references()`](https://inbo.github.io/INBOmd/reference/references.md)
  : Insert references at this position

- [`render_natbib()`](https://inbo.github.io/INBOmd/reference/render_natbib.md)
  :

  Render a Markdown file with `natbib` bibliography

- [`validate_doi()`](https://inbo.github.io/INBOmd/reference/validate_doi.md)
  : validate a DOI
